Configurer Firebase

Firebase est un service gratuit de Google qui permet de synchroniser vos données en temps réel sur tous vos appareils.

Étape 1 — Créer un projet Rendez-vous sur console.firebase.google.com et connectez-vous avec votre compte Google. Cliquez sur Ajouter un projet, donnez-lui un nom (ex: "mes-repas"), désactivez Google Analytics si proposé.
Étape 2 — Créer la base de données Dans le menu gauche, cliquez sur Firestore Database → Créer une base de données. Sélectionnez une région proche (ex: europe-west1).
Étape 3 — Définir les règles de sécurité Dans Firestore Database → onglet Règles, remplacez tout le contenu par :

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
Puis cliquez sur Publier. Ces règles n'expirent pas.
Étape 4 — Récupérer les identifiants Dans les paramètres du projet (⚙️ en haut à gauche) → Paramètres du projet → faites défiler jusqu'à Vos applications. Cliquez sur </> pour ajouter une app Web. Copiez les valeurs apiKey, authDomain et projectId dans les champs ci-dessus.
✅ Gratuit — Le plan Spark (gratuit) de Firebase est largement suffisant pour cet usage. Aucune carte bancaire requise.
